# Frontend Source

This folder contains the SPA source files. Use `src/main.js` as the entrypoint. Organize code into `components/`, `pages/`, and `api/` folders. Keep components framework-free (vanilla JS) and accessible.

Notes:

- Use CSS utility classes from Tailwind; put overrides in `styles/mobile.css`.
- Keep visualizations library-free except Chart.js for scatter plot.
