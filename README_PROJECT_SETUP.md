# Project Setup Checklist

This file contains an actionable checklist for initial project setup.

- [ ] Create project `.env` from `.env.example`
- [ ] Build Docker images and `docker-compose up` (app, web, db)
- [ ] Run database migrations (`database/schema.sql`)
- [ ] Install frontend dependencies and run `npm run dev`
- [ ] Seed initial data (clusters, modules, professors)
- [ ] Create an initial admin user via SQL or seed script
- [ ] Run basic integration tests
